# Portfolio Website

A personal portfolio website built with React, Vite, TypeScript, Tailwind CSS and Motion.

## Features

- Responsive landing page with sections for About, Skills, Projects, Experience, Achievements, and Contact
- Animated UI using `motion`
- Modern React + TypeScript setup with Vite
- Tailwind CSS styling and utility-first design

## Requirements

- Node.js 18 or newer

## Local Development

1. Install dependencies:
   `npm install`
2. Start the local dev server:
   `npm run dev`
3. Open the app in your browser at the local address shown by Vite

## Production Build

- Build the app:
  `npm run build`
- Preview the production build:
  `npm run preview`

## Notes

- No `.env` file is required for this portfolio site
- The project is configured as an ES module app with `type: "module"`
